# Libraries

Common Python libraries used in data science and programming.

## Data Processing

- [[pandas]]: Handle data as tabular tables (like Excel). Supports CSV, Excel, JSON, SQL, and more.
- [[numpy]]: Efficient numerical computation. Handles vectors and matrices.

## I/O & Networking

- [[requests]]: HTTP requests for fetching data from the web.
- [[zipfile]]: Read and write ZIP archives.
- [[io]]: Handle various I/O types (text, binary, raw bytes).
